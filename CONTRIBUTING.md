# Contributing to Tradr

Thank you for your interest in improving Tradr! This document outlines how to contribute while keeping the project’s identity intact.

## Code of Conduct

Be respectful, constructive, and stay focused on building the best terminal trading desk possible. Discriminatory or harassing behavior will not be tolerated.

## Getting Started

1. Fork the repository and create a feature branch.
2. Install dependencies via `uv sync`.
3. Run the app with `uv run tradr-dev` to test your changes interactively.

## Pull Requests

- Keep PRs focused and describe the problem being solved.
- Update tests/docs when applicable.
- Ensure the UI branding (header, footer, `about` command, etc.) still references Wiseman Umanah by default.
- Run `uv build` (or any relevant checks) before opening a PR.
